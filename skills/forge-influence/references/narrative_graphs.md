# Narrative graphs

Narrative as a first-class object in the influence graph.

## Concept

- **Narrative node:** A claim, theme, or story that can spread; linked to entity nodes (origin, amplifiers).
- **Narrative graph:** Subgraph or overlay on the entity graph; edges represent “carries narrative” or “amplifies.”
- **Vector embedding:** Optional; narrative_embedding.py produces embeddings for similarity or clustering.

## Relation to canonical schema

Narrative is a logical type for Forge; map to event or store in Forge outputs (narrative_simulation.json). See lens-intelligence/references/schemas.md for Narrative Object fields.
